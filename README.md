Automated Investment Thesis Generator

Project Overview
The Automated Investment Thesis Generator is an AI-powered web application built using Streamlit. It evaluates startup investment potential based on financial, market, and growth parameters and generates a structured investment thesis report.

Key Features
Interactive startup data input form
Scoring-based investment evaluation system
Risk classification (Low, Moderate, High)
Professional thesis-style report generation
Modern neon glass UI design
Responsive web-based interface

Evaluation Criteria
The system evaluates startups based on:
Annual Revenue
Market Size
Market Growth Rate
Startup Stage
Each parameter contributes weighted points to generate an overall investment score.

Tech Stack
Python
Streamlit
HTML/CSS (custom styling)
GitHub for version control

How to Run Locally
Clone the repository
Install dependencies
pip install -r requirements.txt
Run the application
streamlit run main.py
